# The Home That Grows With You — Phased Owner-Builder Homes

A single-page brochure explaining the owner-builder phased-home business concept:
build a small, permanent, real-foundation home in **planned stages** — start with
only the rooms you can afford today, then grow the same house in place (never a
remodel, never a move) as family and budget grow.

The page covers:

- **Why it exists** — the vanished starter home and the affordability crisis.
- **Two compounding savings** — start small (right-sized footprint) + sweat equity (DIY dial).
- **Watch it grow** — an interactive top-down footprint that steps through Phase 1 → 2 → 3.
- **The mechanism** — engineered *ghost doorways* cast into Stage-1 walls, with a cutaway diagram.
- **Expansion day** — reveal, connect, finish.
- **The ICF wrapper**, the DIY/Pro dial, and the bankability story.

## Tech

A single self-contained [`index.html`](index.html) — no build step, no dependencies.
All diagrams are inline SVG; the only external resource is Google Fonts. Open the
file directly in a browser to view it locally.

## Publishing to GitHub Pages

1. Create a new **empty** repo named `owner-builder-homes` on GitHub (no README/license).
2. Push this repo (see commands below).
3. In the repo, go to **Settings → Pages**.
4. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
5. Choose branch **`main`** and folder **`/ (root)`**, then **Save**.
6. Wait ~1 minute; your site goes live at the URL above.
